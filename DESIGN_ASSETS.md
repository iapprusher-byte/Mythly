## Download Instructions

To download these images:

1. Use `curl` or `wget` to download each image:
```bash
curl -o composeApp/src/commonMain/composeResources/drawable/onboarding_krishna.jpg "<URL>"
```

2. Or manually download from browser and place in the drawable folder

3. After downloading, reference them in Compose using:
```kotlin
painterResource("drawable/onboarding_krishna.jpg")
```
